# Game Addiction Risk Level Finder

This project predicts the risk level of gaming addiction using machine learning.

## Project Overview
This model analyzes various gamer lifestyle factors such as gaming hours, sleep patterns, and social behavior to predict addiction risk.

## Features Used
- Age
- Gender
- Weight Change
- Social Isolation Score
- Face-to-Face Social Hours
- Years of Gaming
- Daily Gaming Hours
- Sleep Hours
- Eye Strain
- Back/Neck Pain

## Machine Learning Model
Algorithm Used:
- Decision Tree Classifier

## Accuracy
Model Accuracy: ~69%

## Files in this Repository
- `app.py` → Streamlit web app
- `DecisionTreeClassifier_model.pkl` → trained model
- `Gaming and Mental Health.csv` → dataset
- `gaming&mental_health_project.ipynb` → training notebook
